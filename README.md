## 학생성적 프로그램을 만들고 관리하기!!
1. 학생정보를 입력, 수정, 검색, 보관, 출력, 정렬, 통계 프로그램을 만든다.
2. MySQL 프로그램을 이용하여 프로시저 함수 및 PK를 활용하여 삭제 및 수정된 내용을 트리거를 사용해 따로 저장한다.
